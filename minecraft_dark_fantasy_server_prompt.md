# Minecraft Dark Fantasy Server Website - Master LLM Prompt

## PROJECT OVERVIEW

Create a modern, high-converting Minecraft server website with a dark
fantasy abyss theme.

Style inspiration: - Dark Souls / Berserk inspired aesthetic - Red
eclipse sky - Glowing demonic eye in background - Lava cracks and
burning textures - Heavy stone UI blocks - Skull icons and medieval
fantasy elements - Cinematic hero section

Design mood: - Dark - Aggressive - Powerful - Abyssal - Cinematic -
Premium gaming brand

------------------------------------------------------------------------

## DESIGN REQUIREMENTS

-   Fully responsive
-   Modern UI/UX
-   Clean spacing and grid layout
-   Dark red + black color palette
-   Glowing red highlights (#ff2a2a, #b30000)
-   Background gradients with subtle smoke/fog effects

------------------------------------------------------------------------

### 1. Features Section

Cards: - Survival - Dungeons - Events

Design: - Stone textured containers - Skull iconography - Lava glow on
hover - Subtle scale animation

### 2. Character Classes Showcase

Include 3-4 RPG style characters: - Abyss Knight - Blood Reaper - Shadow
Berserker - Infernal Guardian

Each card should contain: - Character artwork - Small description -
"View Abilities" button

### 3. Server Stats Section

Animated counters: - Total Registered Players - Discord Members - Server
Uptime - Total Bosses Killed

### 4. Leaderboard Section

Display: - Top 5 players - Minecraft head avatars - XP or kills - Rank
icons - "View Full Leaderboard" button

### 5. Discord CTA Section

Large section with: - Heading: "JOIN THE DARK LEGION" - Big glowing
button - Discord invite link

### 6. Footer

Include: - Home - Store - Vote - Forum - Contact

Dark stone textured background.

## DESIGN SYSTEM

### Colors

Primary Background: - #0a0a0a - #111111

Accent Red: - #ff2a2a - #cc0000 - #b30000

Glow: - rgba(255, 0, 0, 0.6)

### Typography

Headings: - Minecraft-style pixel font OR Cinzel / Medieval serif

Body: - Inter / Poppins / Orbitron

### UI Card Style

background: #141414 border: 1px solid #2a0000 box-shadow: 0 0 15px
rgba(255,0,0,0.2) border-radius: 12px

Hover: transform: scale(1.05) box-shadow: 0 0 25px rgba(255,0,0,0.6)
